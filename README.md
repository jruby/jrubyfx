JRubyFX
=======
JRuby wrapper for JavaFX 2.x.

BUILD
-----
The default ant script expects the JavaFX runtime JAR in build_lib.
It should be found in `$JAVA_HOME/jre/lib/jfxrt.jar` for a suitable
JDK/JRE.

Then, run

    ant jar
